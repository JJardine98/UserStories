User Story Glossary

Priority Level - High

User Story #5 - Crud Operations

Role Want So That
As a Admin To be able to create, read, update and delete entities of relevant data within the software I can easily manage entities and data manually, ensuring quick and simple entries

Acceptance Criteria:

  • Able to login and easily access the areas to update each table.
  • Able to clearly select which operation I wish to execute, and apply it withing the software.
  • Able to create, read, update, delete all information for the following tables: Employees, Locations, inventory, suppliers, permissions, and employee permissions.
