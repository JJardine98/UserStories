Priority Level - Medium

User Story #11 - Warehouse Returns

Role Want So That
Warehouse Foreman To be able to process any losses, returns, and damaged goods, and to be able to decide if a supplier order is adequate for acceptance. Any accepted, returned, or damaged goods can be tracked within the system.

Acceptance Criteria:

  • To be able to input any losses or returns when receiving from supplier.
