User Story #21 - Manager Generate Reports

User Story Glossary

Priority Level - High

User Story #21 - Manager Generate Reports

Role Want So That
As a Store Manager To be able to generate reports in fields relevant to my job. I can easily access data reports in different fields and print them.

Acceptance Criteria:

  • Able to easily access the report management screen.
  • Able to create reports for at least the following aspects: Store orders, inventory, losses, sales, and trends.
  • Able to create the reports based on selected date ranges.
  • Able to generate reports based on all locations or a selected location.
