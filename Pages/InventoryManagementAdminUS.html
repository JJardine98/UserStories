User Story Glossary

Priority Level - High

User Story #4 - Inventory Management (Admin / Manager)

Role Want So That
As a Admin / Manager Search Items Search Items by UPC, Name, Category, Bardcode Scanning or Item ID
As a Admin / Manager Edit Quantities Edit Quantities of items when recieved
As a Admin / Manager View Inventory Quickly and efficiently view inventory levels
As a Admin / Manager Add New Inventory Add new inventory to the system
As a Admin / Manager Edit New Inventory Manage new inventory items

Acceptance Criteria:

  • Admins and Managers for items by UPC, name, category, barcode scanning, or item ID.
  • Admins and Managers can edit the quantities of items when received.
  • Admins and Managers can view the inventory of all items.
  • Admins and Managers can add new inventory items.
  • Admins and Managers can edit new inventory items.
