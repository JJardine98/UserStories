User Story Glossary

Priority Level - Medium

User Story #23 - View Orders / Shipping Receipts

Role Want So That
As an Acadia Trucking Employee To be able to view the orders we are receiving as well as the shipping receipts for completed orders. We can track how things are going and plan accordingly for future orders.

Acceptance Criteria:

  • Must be able to view by individual locations or all locations.
  • Able to view currently outstanding or by date range for both.
  • Both must include size and weight details.
