Priority Level - High

User Story #17 - Handling Inventory

Role Want So That
As a Store Manager To be able to view, edit, move, and perform inventory within the system. I can easily handle all store inventory and ensure all information is accurate.

Acceptance Criteria:

  • Able to view inventory for location and other locations.
  • Able to edit inventory descriptions, min reorder size, and notes.
  • Perform inventory at location.
  • Move inventory to new locations, or assign as lost or damaged.
  • Receive new inventory.
