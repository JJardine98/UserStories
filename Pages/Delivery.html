User Story Glossary

Priority Level - Medium

User Story #22 - Generate Shipping Receipt

Role Want So That
As a Delivery Driver To be able to generate a shipping receipt for a delivery. Both parties have a copy of a delivery transaction with relevant information included.

Acceptance Criteria:

  • Able to easily access the report management screen.
  • Items are listed by pallet.
  • Vehicle size is listed.
