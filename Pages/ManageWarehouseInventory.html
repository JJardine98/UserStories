User Story #8 - Manage Warehouse Inventory

User Story Glossary

Priority Level - High

User Story #8 - Manage Warehouse Inventory

Role Want So That
Warehouse Foreman To be able to view inhouse and instore inventory, edit inventory, and change locations of inventory. Changes in inventory can be easily tracked and updated to the system, minimizing mistakes and delays.

Acceptance Criteria:

  • Easily able to access the inventory management system.
  • Able to view inventory at all locations.
  • Edit or add inventories description, case size, minimum reorder size, and notes.
