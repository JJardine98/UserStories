User Story Glossary

Priority Level - High

User Story #16 - Create Store Order / Emergency Order

Role Want So That
As a Store Manager To be able create weekly store orders and emergency orders from within the system. I can easily align my inventory needs with items entered into the orders, and have them sent directly to the warehouse foreman.

Acceptance Criteria:

  • Can easily access the orders management page.
  • Can decide to create a store order or an emergency order.
  • Can view warehouse inventory while ordering.
  • Automatically add items that have fallen below the minimum reorder quantity.
  • Orders by case size.
  • Has an option to input notes.
  • Can be revisited and edited prior to submitting.
  • Max 5 line items for emergency orders.
  • Only 1 outstanding order of each type permitted at a time.
