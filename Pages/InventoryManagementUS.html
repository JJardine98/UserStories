User Story #3 - Inventory Management (Warehouse Worker / User)

Role Want So That
As a User Search Items Search Items by UPC, Name, Category, Bardcode Scanning or Item ID
As a Warehouse Worker Edit Quantities Edit Quantities of items when recieved
As a User / Warehouse Worker View Inventory Quickly and efficiently view inventory levels

Acceptance Criteria:

  • Users should be able to search for items by UPC, Name, Category, Bardcode Scanning or Item ID.
  • Warehouse workers should be able to edit the quantities of items when received.
  • Users and Warehouse workers should be able to view the inventory of all items.
