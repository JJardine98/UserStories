User Story Glossary

Priority Level - Medium

User Story #15 - Warehouse Perform Inventory

Role Want So That
Warehouse Foreman To be able to perform inventory on location. Inventory Data can be efficiently verified in the system.

Acceptance Criteria:

  • Able to scan items to confirm it's attendance in the inventory.
  • Able to save inventory check into the system.
