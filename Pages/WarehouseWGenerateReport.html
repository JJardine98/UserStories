User Story #14 - Warehouse Worker Generate Report

User Story Glossary

Priority Level - Medium

User Story #14 - Warehouse Worker Generate Report

Role Want So That
Warehouse Foreman To be able to generate reports relevant to my position. I can easily access data reports in different fields and print them.

Acceptance Criteria:

  • Able to easily access the report management screen.
  • Able to create reports for at least the following aspects: Store orders, inventory, Losses, Shipments from suppliers, emergency orders, and sales trends.
  • Able to create the reports based on selected date ranges.
  • Able to generate reports based on all locations or a selected location.
  • Shipping receipts listed by pallets.
  • Have a default sorting for shipping orders, prioritizing outstanding orders.
