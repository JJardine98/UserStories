User Story Glossary

Priority Level - High

User Story #18 - View Orders

Role Want So That
As a Store Manager To be able to view store orders, order status, as well as back orders. I can be informed on the status of inventory I will be receiving and when.

Acceptance Criteria:

  • Can review store orders, default local, all and data range by selection
  • Can view store order status for each store order
  • Can view backorders, default local, all and data range by selection
